# Portfólio Pessoal

Este é o meu site pessoal, desenvolvido do zero com **HTML, CSS e JavaScript**,
com o objetivo de apresentar meus projetos, habilidades e certificados,
além de testar e aplicar boas práticas de desenvolvimento front-end.

## Funcionalidades

- Navegação dinâmica entre seções (SPA)
- Persistência da seção ativa usando `localStorage`
- Animações com GSAP
- Layout responsivo
- Organização modular de código

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- GSAP (animações)
- Git & GitHub
